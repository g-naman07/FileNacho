# File Nacho Frontend

Frontend for `File Nacho`, a peer-to-peer file sharing app built with React and Vite.

## Environment

Create `frontend/.env` from the example file and set:

- `VITE_SIGNALING_SERVER_URL=ws://localhost:5000` for local backend
- `VITE_SIGNALING_SERVER_URL=wss://your-backend-domain.com` for production

## Run

```bash
npm install
npm run dev
```
